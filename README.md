# problem-spark-engine

> Discover real problems people are talking about online.

![Status: Experiment](https://img.shields.io/badge/Status-Experiment-yellow)

## What It Does

Aggregates problems and pain points from Reddit, Hacker News, and Twitter. Search, filter by tags and source, bookmark ideas, and find inspiration for your next project. Includes web scraping triggers and falls back to mock data when the API is unavailable.

## Tech Stack

`React` `Vite` `TypeScript` `Supabase` `shadcn/ui` `Tailwind CSS` `Axios`
